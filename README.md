# DotClaude

Este projeto compreende:

- Arquivos para o diretório .claude
- Arquivos e instruções agnósticas de projetos
- Esqueleto para se colocar instruções para linguages, tecnologias, stack, etc

## Diretório `local`

Instruções locais específicas podem ser colocadas no diretório `local`, que não está no controle de versão.

## Arquivo `settings.local.json`

Como o arquivo `settings.local.json` pode ser muito específico do projeto, ele não está no control de versão e é usado um arquivo `settings.local.example.json` no lugar.

## Subtree

Recomenda-se criar uma subtree do git deste repositório no projeto em que este será usado.

## CLAUDE.md

Há uma amostra de um arquivo CLAUDE.md (CLAUDE.example.md) neste repositório que pode servir de base para um do projeto.
